## 🌿 Sobre o Projeto

O **BioUrbano** nasceu como uma iniciativa voltada para o reflorestamento e conscientização ecológica dentro das cidades. O ecossistema urbano sofre diariamente com a falta de biodiversidade vegetal. Este catálogo digital oferece fichas técnicas detalhadas sobre espécies que se adaptam muito bem ao ambiente urbano, promovendo benefícios essenciais como o controle de erosão, o aumento de áreas de sombra natural, o sequestro de carbono e o fornecimento de habitat para a fauna local.

O projeto é inspirado na flora urbana mapeada na região de **Cornélio Procópio, Paraná**.

---

## ⚙️ Funcionalidades

- **Catálogo Interativo:** Cards detalhados com imagens reais de espécies adequadas para o plantio urbano (como Pitangueira, Jabuticabeira, Ora-pro-nobis, Goiabeira, entre outras).
- **Fichas Técnicas Integradas:** Exibição ágil do porte/tamanho da árvore, tipo ideal de solo e lista de benefícios ecológicos práticos de cada planta.
- **Sistema de Autenticação Visual:** Telas estruturadas para o "Registro" e "Acesso de Usuários", prontas para integração com o ecossistema Back-end.
- **Responsividade Total:** Interface adaptável (`@media queries`) que garante uma excelente navegação em computadores, tablets ou smartphones.

---

## 🗂️ Layout e Páginas

O projeto encontra-se modularizado e dividido em arquivos estáticos de fácil manutenção:

1. **`index.html` (Catálogo Principal):** Apresentação do projeto ODS 15, banner informativo e a listagem em grid dos cards botânicos com fotos de perfil de cada árvore.
2. **`cadastro.html` (Portal do Voluntário):** Contém formulários limpos e estilizados para criação de novas contas de voluntários mapeadores e área de login integrado.
3. **`curriculo.html` (Portfólio do Integrante):** Página integrada contendo o perfil profissional, objetivos ODS associados e competências técnicas do desenvolvedor responsável.
4. **`style.css` (Identidade Visual):** Centralização de variáveis CSS (`:root`) com uma paleta de cores voltada à natureza (`--forest-green`, `--olive`, `--earth-brown`, `--ivory`).

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas de desenvolvimento Web Semântico e Design Responsivo:

- **HTML5:** Utilização de tags semânticas estruturais (`<header>`, `<main>`, `<section>`, `<article>`, `<figure>`, `<nav>`, `<aside>`).
- **CSS3:** Estilização baseada em variáveis nativas, layouts em **CSS Grid** para os catálogos e **Flexbox** para componentes de navegação, além de animações de transição suave (`transition`) nos cards.
- **Google Fonts:** Tipografia elegante mesclando as fontes *Merriweather* (títulos de forte presença) e *Poppins* (leitura limpa e moderna).
- **Material Icons Outlined:** Pacote de ícones minimalistas para links de dados e de contato.
